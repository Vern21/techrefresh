How To Do A Tech Refresh

By: LeVern Roberts

What is a Tech Refresh

A tech refresh is when a user is eligible to switch out their old computer for a new one.

The process

  • Before we can start the refresh the user has to put in a requets that has to be processed
  • They first have to see if the warranty has expired
  • They have to choose if they want a Lenovo X1 carbon 3rd gen or 13" retina display macbook pro

Lenovo Imaging

  • Make sure the computer is hardwire in
  • Follow the steps on the PC imaging sheet
  • Login into Landesk
  • imaging takes 1-2 hours to complete
  • Run the post image sricpt, the printer sricpt
  • Then we upload load their previous data

Macbook Pro Imaging

Transfering Data
